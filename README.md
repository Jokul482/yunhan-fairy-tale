# 小鹿妈妈养育帮手

专为新手父母打造的宝宝日常记录小工具。提供喂养计时、尿布记录、睡眠定时、护眼模式、睡前仪式清单、疫苗参考等功能。所有数据保存在手机本地，无需注册登录。

## 功能概览

### 记录 Tab
- **喂养记录**：母乳计时器 + 奶粉毫升数快速记录，今日汇总
- **尿布记录**：湿 / 脏 / 都有，一键记录
- **心情记录**：Emoji 心情选择，简单记录妈妈状态
- **统一时间轴**：喂养 + 尿布 + 心情按时间轴混合展示，左滑删除

### 睡前 Tab
- **睡眠定时器**：15/30/45/60 分钟预设，计时结束震动提醒
- **护眼模式**：暖色柔光遮罩，减少睡前蓝光刺激
- **睡前仪式清单**：刷牙洗脸换睡衣等 7 个默认事项，支持自定义

### 百科 Tab
- **成长日历**：输入宝宝生日，自动计算天数、月龄，倒数重要里程碑
- **疫苗接种参考**：中国国家免疫规划程序表，按年龄分组展示，可勾选已完成
- **发育里程碑**：按月龄分 4 个维度（大运动/精细运动/语言/社交），可勾选已观察到

## 技术栈

- **框架**：uni-app (Vue 3 + TypeScript + SCSS)
- **平台**：微信小程序
- **存储**：纯本地存储（wx.Storage），无后端/无云服务

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发模式（微信小程序）
pnpm dev:mp-weixin

# 用微信开发者工具打开 dist/dev/mp-weixin 目录
```

## 编译发布

```bash
# 构建微信小程序
pnpm build:mp-weixin

# 在微信开发者工具中打开 dist/build/mp-weixin，点击上传
```

## 项目结构

```
src/
├── pages/
│   ├── logs/logs.vue      # 记录 Tab（喂养/尿布/心情/时间轴）
│   ├── sleep/sleep.vue     # 睡前 Tab（定时器/护眼/清单）
│   └── tools/tools.vue     # 百科 Tab（成长日历/疫苗/里程碑）
├── styles/
│   └── variables.scss      # 全局样式变量（颜色/圆角/字号/间距）
├── utils/
│   ├── constants.ts        # 常量（存储 Key / 疫苗数据 / 里程碑数据）
│   ├── storage.ts          # 存储工具（读写封装 / 30天自动清理）
│   └── sys.ts              # 微信更新管理器
├── pages.json              # 页面路由 + Tabbar 配置
└── manifest.json           # uni-app 应用配置
static/
└── tab/                    # Tabbar 图标
```

## 审核注意事项

- 微信后台服务类目请选 **工具-健康管理**
- 疫苗/里程碑页面有免责声明，不提供任何医疗建议
- 无用户生成内容、无社交功能、无付费功能
- 所有数据纯本地存储
